# 定义类

```python
class Dog():
	"""类描述"""
	
	def __init()__(self):
		something();
```

1. `__init()__` 类似 Java 的构造方法
2. `self` 类似 Java 的 `this`，由 Python 解释器自动传入
3. 成员方法第一个参数必须是 `self` 且在调用时不用传入
4. **与静态语言不同，Python 允许运行时添加属性**

# 实例化对象

```python
var = Dog()
```
