JVM 运行时数据区域：

- 线程私有
    - 程序计数器
    - Java 方法栈
    - 本地方法栈
- 线程共享
    - 方法区：存储被 JVM 加载的类信息（Class 文件信息）、常量、静态变量。
        - 运行时常量池：是方法区的一部分，存放 Class 文件的常量池表信息。
    - 堆

Class 文件有什么？

类中的字段、方法、接口等信息，还有常量池表（存放该类编译时生成的字面量、[[符号引用]]）
