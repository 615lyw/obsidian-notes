# 方法签名

方法名称和参数列表的组合。

# 方法重写 Override

子类可以重写父类的方法。

要求：

- 访问权限修饰符限制
- throws 异常限制
- 返回值限制
- 方法签名必须相同

子类重写该方法时 throws 异常必须是父类声明异常的子集：

- 子类不 throws 异常
- 子类 throws 父类的部分异常
- 子类 throws 父类异常的子类
- 子类 throws 父类的异常

不能：

- 子类 throws 父类异常的父类
- 子类 throws 父类未声明的异常

# 方法重载 Overload

一个类可以有多个同名方法，只要他们的方法签名不同即可。
