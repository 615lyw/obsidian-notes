# 程序、进程与线程

- 程序是指令、数据的集合，是静态的
- 进程是程序的运行态，是操作系统资源分配的基本单位
- 线程：轻量级进程，是程序执行的最小单位
    - 特点：
        - 线程间共享父进程的资源
        - 线程间切换开销比进程低
        - 线程间通信开销比进程间通信低
