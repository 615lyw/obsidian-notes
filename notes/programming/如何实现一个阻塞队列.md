解决方案模型：[[管程模型]]。

- 两个核心操作：入队和出队
- 一个存储结构：队列

出队要求**队列不空**，入队要求**队列不满**

队列不空和队列不满即两个条件变量 A，B

```
入队：
if (队列已满)
    B.wait()
// 入队操作
A.notify()

出队：
if (队列为空)
    A.wait()
// 出队操作
B.notify()
```

入队和出队方法保证互斥，还需要一把互斥锁。
