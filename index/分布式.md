# 概述

分布式系统由一组为了完成共同任务而协调工作的计算机节点组成，它们通过网络进行通信。

常见切分法：

- 水平切分：将同一系统部署到多个节点上
- 垂直切分：将系统按业务进行切分，各业务服务部署在不同的节点上，多个节点协作共同完成业务请求
- 混合切分

面临的问题：

- 数据一致性问题
- 可拓展性
- 高性能
- 可用性
- 大数据等等
